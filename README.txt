Installation profile for the Wizzlern Drupal Architecture training.
See http://wizzlern.nl/training/drupal-architectuur

Copy the files in the Drupal profiles directory into the folder:
  profiles/wizzlern_architect

Start the Drupal installation in your browser. In the first step of the
installation select the "Wizzlern Drupal Architect" installation profile.
Proceed as usual.
